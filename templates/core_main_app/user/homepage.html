Find Data and Information About the Circular Economy

This registry is a tool for finding data and information resources related to enabling the US Circular Economy (CE). It is part of an initiative at the National Institute of Standards and Technology (NIST) to support the Nation's need to transition away from a linear--take, make, use, waste--economic model toward one in which materials repeatedly cycle within the economy.

With this registry, one can search for resources based on keywords and filter the results based on CE concepts and topics. The types of resources that can be found here are listed at the right. Each resource description includes a link to where it can be found on the web.

The content of this registry is curated by the Circular Economy Initiative at NIST. For more information about this registry or how to contribute to it, please contact us at ceregistry@nist.gov.
